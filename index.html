üìù Cara Penggunaan

Import komponen di HTML:

<script type="module" src="./components/my-card.js"></script> <script type="module" src="./components/my-button.js"></script> <my-card title="Hello World" theme="primary"> <p>Konten card di sini</p> </my-card>

Manipulasi komponen via JavaScript:

const card = document.querySelector('my-card'); // Update via attribute card.setAttribute('title', 'New Title'); card.setAttribute('theme', 'success'); // Update via public API card.setTitle('Another Title'); card.setTheme('dark');
